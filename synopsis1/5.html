System Module & Their Functionalities

1. Admin module

Role : System Supervisior

Key Functionalities
  • manage all user(Student,Mentors,Companies)
  • Approve /Reject new company add mentor registrations.
  • view system-wide analytics(applications,tasls,completion)
  • Handle report and escalate unresolved issues.
  • Backup and export data(CSV ,PDF,report.
2. Company /HR Module

Role: internship provider

Key Functionalities
  • register and manage company profile .
  • create and managing internship listing (roles ,stipend,duration)
  • view student application and shortlist candidates
  • Assign mentors to select interns.
  • create task and allocate them with deadlines
  • Evaluate interns with feedback and performance ratings.
  • provide feedback and score task
  • Receive feedback and download internship certificate.
4.Student Module

Role: intern / User

key functionalities
  • register and complete profile(skills,resume etc)
  • search and apply for internship
  • view application status(applied/shortlisted /selected)
  • Receive and submit tasks.
  • upload weakly progress report.
  • receive feedback and download internship certificate

6. Key FEature

  • Role-based Access Control
  • Dynamic internship listing & Fibers
  • application Workflow management
  • Task assignment with due dates
  • progress tracking & dashboards
  • weekly Report upload system
  • intern evaluation & scoring
  • Auto certificate Generation(PDF)
  • secure Authentication with JWT
  • Email Notification & alert
